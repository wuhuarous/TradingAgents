# 界面与选股优化说明

## 1. 本轮目标

本轮把原有页面从“单点功能”优化为“模拟交易工作台”：

- 仪表板：看年化 50% 目标进度、全市场 Top10、持仓和最近训练复盘。
- 行情总览：看 A股、港股、美股市场强弱，以及全市场强势候选。
- 智能选股：默认展示全市场 Top10 优质股和 Top10 潜力股。
- 新闻资讯：展示市场情绪均值、积极/中性/消极新闻数量和过滤列表。
- 深度分析：支持从全市场候选一键带入标的，执行多 Agent 深度分析。

## 2. 全市场 Top10 选股

接口：

```text
GET /api/simulation/rankings?limit=10
```

返回：

- `quality_top10`：偏质量、成长、估值、低风险。
- `potential_top10`：偏行情动量、成长、新闻情绪、流动性。
- `markets`：各市场候选概览。
- `all`：综合分候选池。

当前覆盖：

- A股。
- 港股。
- 美股。

## 3. 性能边界

A股候选会优先使用当前行情、财务、技术和新闻接口计算。

港股/美股在全市场榜单中使用快速候选画像，原因是海外数据源可能限流或响应较慢。用户点击进入深度分析后，再刷新实时行情、财务和新闻。

下一步可优化为：

- 为港美股增加稳定付费数据源。
- 将候选评分写入缓存。
- 后台定时刷新全市场榜单。
- 前端只读取最近一次刷新结果。

## 4. 下一步建议

优先级最高的是把 Top10 结果从“即时计算”升级为“定时任务 + 缓存结果”：

```text
定时刷新候选池
  -> 保存评分快照
  -> 前端读取快照
  -> 模拟交易使用快照
  -> 复盘回填真实收益
```

这样页面会更快，复盘数据也更稳定。
